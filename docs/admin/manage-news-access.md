# Manage Authors

Authors are users who can create and manage news content in the app and web. To grant a user Author privileges and enable the Create News and Manage News features, follow these steps in the administrative console:

Navigate to the Author section in the admin panel.

[![Author Page](/images/panel/author-page.png)](/images/panel/author-page.png)

Go to Author Tab > Operate, and set the user's status to Approved.

Note: Setting the status to Approved unlocks these capabilities instantly. The "Create News" and "Manage News" options will then be visible to the user under the Profile Settings screen on both the mobile application and the web platform.

[![Author Status](/images/panel/author-status.png)](/images/panel/author-status.png)
