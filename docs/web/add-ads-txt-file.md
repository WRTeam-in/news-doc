# How to Add ads.txt file

## With SEO

1. Create a ads.txt file in public folder
2. Paste your ads.txt code in it.
3. Upload your new code to your server.
4. Rebuild your project.
5. Restart your project by `pm2 restart your_project_id`

## Without SEO

1. Create a ads.txt file in root folder
2. Paste your ads.txt code in it.
3. Rebuild your project.
4. Upload your new code to your server.
